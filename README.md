# AI-Based Vendor Risk Scoring System

## Project Overview
This project aims to develop an AI-Based Vendor Risk Scoring System to assess and score vendors based on various risk factors, enabling businesses to make informed decisions when engaging with vendors.

## System Architecture
The system is structured into three layers:
1. **Vendor Scorecard**: A framework to evaluate vendor information and risk factors.
2. **ML Prediction**: A machine learning model that predicts the risk score based on historical data.
3. **NLP News Risk**: An NLP component that analyses news articles to identify potential risks associated with vendors.

## Required Dependencies
To run the system, the following dependencies are required:
- Python
- Pandas
- Scikit-learn
- Matplotlib
- openpyxl

## Instructions for Running the System
1. Install the required dependencies:
   ```bash

   ```
